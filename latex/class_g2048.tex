This function starts/resumes a game of 2048. For as long there are valid moves, the games continues until the player decides to quit or there are no moves left. Every loop the program asks for an input. If the player gives a valid move, \hyperlink{class_g2048_af65fedafade111894861afd881aa054d}{add\+Tile()} is called. When the loop is broken, \hyperlink{class_g2048_a335bb5d25c76f2e2548115e64091afc0}{save\+Game()} is called and the program ends.
